SMERFS Sentiment Analyzer

SMERFS, The Social Media Emotional Response Feedback Software, is my fourth year University project, built on the Django Rest Framework, and ReactJS. An individual project which involved building an API endpoint and a User Interface from the ground up, allowing multiple users to generate dashboards of user sentiment analysis, conducted on social media posts in relation to a given topic. Users would be presented with a screen for entering text input, that input would be used to pull down tweets from twitter. Those tweets would then be translated and cleaned before being ran through a Sentiment Analysis model. The analyzed posts would then be gathered, and compiled into a visualization dashboard of information using Tableau Server. Some visualizations would include time data, to show how sentiment changes over time, geographical data, to show how sentiment changes by region, and general sentiment data, which would show how sentiment varies for a given topic.

Technologies & Tools Used

Python Django REST JavaScript ReactJS NodeJS VADER Sentiment PostgreSQL
